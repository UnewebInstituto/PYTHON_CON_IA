CLASE: Es la entidad de programación fundamental en la POO. Combina declaración de 
variables, que se conocen como atributos, con funciones que se conocen como métodos.

Por ejemplo:

    #atributos
    nombre=''

    #métodos
    def caminar(self):
        return 'La persona camina'

    def comer(self):
        return 'La persona come'

    def dormir(self):
        return 'La persona duerme'
